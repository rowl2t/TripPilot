# Web Landing (apps/web)

## 목표
- 출시 전후 마케팅/신뢰 확보
- SEO 친화 구조
- 법적 문서 포함

## 페이지
- `/` Hero, 문제/해결, 핵심 기능, 가격, FAQ, 다운로드/대기자 CTA
- `/pricing` Free/Pro/Trip Pack
- `/blog` 정적 콘텐츠 구조
- `/legal/terms`, `/legal/privacy`, `/legal/disclaimer`
- `/support` 문의 폼(mock)

## SEO
- `app/layout.tsx` metadata + OpenGraph placeholder 설정
- 정적 콘텐츠 우선 렌더링으로 색인 최적화

## 디자인
- 모바일 우선 레이아웃
- 신뢰 중심 톤
- 앱 스크린샷 섹션은 추후 실 이미지 교체
