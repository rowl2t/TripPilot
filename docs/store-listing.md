## Full description
TripPilot helps first-time and busy travelers build complete travel plans from a few inputs. 
Create AI-powered itineraries, generate booking checklists, save travel links from social media, and organize plans with companions.

Key features:
- AI itinerary generation with structured day-by-day plans
- Booking task checklist with reminder-oriented guidance
- Saved link analysis from public metadata and user-provided content
- Collaboration features for trip companions
- Offline pack and trip review utilities

Notes:
- Availability/pricing on external booking sites may change.
- Some features require network and region-specific service availability.
