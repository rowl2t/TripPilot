# Onboarding & First-Trip Conversion

## 목표
- 설치 직후 첫 여행 생성까지의 전환율 개선
- 입력 부담을 줄이고 가치 전달을 앞당김

## Onboarding 개선
- 화면 수를 3개로 제한
- 핵심 가치만 전달
  - AI 전체 일정 설계
  - 예약 체크리스트 자동 정리
  - SNS 저장 장소 반영
- 마지막 화면 CTA: `첫 여행 만들기`

## First Trip Flow 간소화
- 최소 입력 모드
  - 여행지
  - 날짜
  - 예산
  - 스타일
- 세부 조건은 Advanced 섹션으로 숨김
- 로그인 전 체험 가능 안내 문구 추가

## Progress UX
- `1분 안에 초안 만들기` 메시지 상시 노출
- 생성 단계 안내
  - 입력 검증
  - 장소 후보 수집
  - 일정 초안 생성
- 생성 중 이탈 시 홈/알림에서 확인 가능 안내

## Empty Home
- 첫 여행 만들기 CTA
- 예시 여행 카드
- 저장한 SNS 링크 추가 CTA
