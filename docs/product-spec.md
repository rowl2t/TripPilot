# TripPilot Product Spec

## 서비스 개요
TripPilot은 여행 초보자가 최소 입력으로 일정, 체크리스트, 예약 가이드를 자동 생성하는 AI 여행 어시스턴트다.

## 핵심 사용자
- 첫 해외여행/자유여행 사용자
- 여행 정보를 SNS에서 수집하지만 정리 어려운 사용자
- 일정 관리와 예약 준비에 부담을 느끼는 사용자

## 핵심 문제
- 정보 과잉으로 인한 의사결정 피로
- 일정/예약/준비물 분절 관리
- 여행 콘텐츠 저장 후 재활용 실패

## 핵심 기능
- AI 여행 일정 생성 및 일자별 플랜
- 예약 체크리스트 자동 생성
- SNS 링크 저장 및 장소 추출/정규화
- 장소 검증(Places API) 기반 신뢰도 강화

## 추가 기능
- 동행자 투표: 후보 일정/장소에 대해 그룹 투표
- 오프라인 여행팩: PDF/오프라인 지도/긴급 연락처 번들

## 수익 구조
- Freemium + Pro 구독(RevenueCat)
- 웹 확장 결제(Stripe)
- 제휴 딥링크/추천 수익

## 비기능 요구사항
- 99.9% 가용성 목표
- 핵심 API p95 400ms 이하(캐시 적중 시)
- 개인정보 암호화 및 접근통제
- 관측성(로그/메트릭/트레이싱)

## 보안 원칙
- 비밀값 환경변수 관리
- 최소권한 IAM
- 입력 검증(Zod)
- 감사 로그 및 이상행위 탐지

## AI 사용 원칙
- 구조화 출력(JSON Schema/Zod) 강제
- 환각 완화: 외부 검증 데이터 결합
- 민감정보 최소 제공

## 예약/제휴 정책
- 초기 직접 결제 미지원
- 제휴 딥링크, 가이드, 리마인더 중심
- 파트너 정책/표시 의무 준수

## SNS 링크 분석 정책
- 공개 메타데이터, 사용자 제공 콘텐츠, 허용 API만 사용
- 로그인/DRM 우회, 무단 스크래핑 금지
