## 이번 수정 사항
- Profile 화면에서 런타임 타입 오류를 유발하던 임시 mock-client 직접 호출 제거
- 서버 연결/로그인 필요 액션은 사용자 친화 가드 메시지로 전환

## 발견 이슈 분류

### Blocker
1. **CI 검증 불가**: `pnpm` bootstrap 네트워크 403으로 `lint/typecheck/test/build` 실행 불가.
2. **실서비스 연동 검증 미완료**: 결제/지원/프라이버시 요청의 실제 backend round-trip 미확인.

### Non-blocker
1. 로그인/회원가입 화면이 아직 단순 스켈레톤.
2. 일부 기능은 mock provider 전제(결제/외부 API).
3. 운영 대시보드는 helper 중심이며 실제 admin UI 통합은 후속 작업 필요.

## 권고 액션
1. CI runner에서 pnpm bootstrap 네트워크 문제 우선 해결.
2. staging 환경에서 15개 플로우 E2E 리허설 재실행.
3. mock/provider 플래그를 환경별로 명시하고 prod 빌드에서 mock 차단 점검 추가.
