# Manual QA Checklist

1. 신규 가입
2. 온보딩
3. 새 여행 생성
4. AI 일정 생성 (mock/production)
5. 여행 상세 확인
6. 장소 옵션 선택
7. 예약 체크리스트 완료/스킵
8. 캘린더 export
9. SNS 링크 저장/분석
10. 장소 후보 확정
11. 동행자 초대
12. 투표
13. 일정 재생성
14. paywall 진입
15. mock 구독 활성화
16. 오프라인 여행팩 생성
17. 문의 제출
18. 데이터 export
19. 계정 삭제 요청
20. 네트워크/API 장애 fallback 확인

각 항목은 Loading/Error/Empty 상태, 라우팅, 권한, 안내 문구까지 확인한다.
