# 수동 QA 체크리스트

필수 플로우(20개):
1. 온보딩 첫 진입
2. 로그인/로그아웃
3. 여행 생성
4. AI 일정 생성(mock/production)
5. 일정 상세 진입
6. 일정 수정/저장
7. 저장된 링크 추가
8. 캘린더 내보내기
9. 오프라인 팩 다운로드
10. 오프라인 상태 재진입
11. 협업 초대 발송
12. 협업 수락/거절
13. 알림 권한 요청
14. 페이월 진입
15. mock 구독 활성화
16. 결제 복원
17. 프로필 수정
18. 데이터 내보내기
19. 계정 삭제 요청
20. 네트워크/API 장애 fallback 확인

각 항목은 Loading/Error/Empty 상태, 라우팅, 권한, 안내 문구까지 함께 점검한다.
